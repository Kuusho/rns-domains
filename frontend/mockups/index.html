RNS · Frontend track · Brand direction selection

Three directions for the most interesting name service registration app.

Each mockup contains the same four moments: hero search, name-available state, the 60-second commit-reveal wait, and a profile view. Click through all three before forming an opinion — the cumulative feel matters more than the first impression.

3 directions · same content · viewable side-by-side
01Solar Genesis

Treats every registration like a sunrise. Editorial serif, warm-to-cool gradient palette, reverent copy, slow rising motion. Differentiates by emotional weight — most crypto apps are cold; this one is almost spiritual.

Type
Newsreader · Inter · JetBrains Mono
Voice
Reverent, editorial, "the first light"
Motion
Slow rises, dawn gradients, no overshoot

02Brutalist Protocol

Anti-app. Terminal aesthetic, all monospace, single orange accent, zero decoration. Differentiates by refusing friendliness — looks like a system tool, not a startup. Status-bar nav, terminal output, record-block forms.

Type
JetBrains Mono only (all sizes)
Voice
Terse uppercase, command-line, "STATUS: AVAILABLE"
Motion
Cuts, blinks, no eases (steps only)
Open mockup →
03Agentic Field

Built for humans AND AI agents. Neon void palette, variable display sans, live data streams, glowing edges. Differentiates by acknowledging the non-human audience — pulsing mempool feed, live block counter, gradient glows.

Type
Big Shoulders Display · Sora · JetBrains Mono
Voice
"For humans and the things they build."
Motion
Pulses, drifts, live streams, glow filters
